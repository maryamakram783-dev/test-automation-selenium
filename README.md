## ⚙️ How to Run

1. Install Java JDK 11+
2. Download ChromeDriver matching your Chrome version
3. Add Selenium and TestNG dependencies
4. Place chromedriver.exe in `drivers/` folder
5. Run `LoginTest.java` via TestNG

---

## 📚 Key Learnings

- How to locate web elements using By.id, By.name, By.xpath
- Using WebDriverWait for dynamic elements
- Writing assertions with TestNG Assert
- Setting up and tearing down browser sessions
- Handling test execution and reading reports
